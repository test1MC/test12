Image Studio Software Setup Wizard

  1. Open Win Image Studio Installer 5.x.x on your computer.

  2. Follow instructions in the Setup Wizard.

    After the Image Studio installation process is completed, installation of the communication program (mDNSResponder) will begin automatically. If upgrading from a previous version of Image Studio, this installation may not be needed.

  3. Please refer to instructions included with your additional application keys, if applicable, for new installations.

    If you are upgrading from a previous version of on a Mac, you will need to reinstall additional application keys.
