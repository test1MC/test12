Introduction

The application is a data analysis package for instruments, including the Odyssey family, Pearl family, and the . These imagers acquire images for use in the Life Sciences, including Western blots, gels, DNA gels, microplates, arrays, and more. Image Studio Software sets the acquisition parameters for the imager, organizes the acquired images in a table, and analyzes the data on the images.
