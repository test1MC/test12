Recommended Methods for Analyzing Plates and Grids

Use the following table as a quick reference for which background method to use when analyzing microwell plate experiments or micro or macroarray experiments.

The background methods described below are available with specific software keys (please visit www.licor.com/AnalysisKeys for information about key availability).The Changing the Background Method section on page 1 describes where applications can be found, if they are installed.
