Print Single Image Per Page or Lab Book Reports

The two printing options are based on Lab Book layouts. SeeLab Book for how to edit layouts.

 

Single Image Per Page: Prints the image or images currently selected in the using the Single Image Per Page Lab Book format, even if another Lab Book format has been applied to the image. The following will be included in the print-out:

  • The image
  • Acquisition Information (Image ID, Image Name, Image Modifications, Project, etc.)
  • Image Display Values (Channel, Color, Resolution, Scan Height, Scan Width, etc.)
  • Header

Lab Book: Prints the image or images currently selected in the using the current Lab Book format.
