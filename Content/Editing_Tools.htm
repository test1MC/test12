Editing Tools

There are several tools for editing features on an image. To edit one or more features, first select the feature(s).

Selecting a feature

  1. Click the button .
  2. Click anywhere inside the feature that you want to select.
  3. To extend the selection (select another feature), hold down and click anywhere inside the next feature you want to select.

You can also do a "sweep" selection by dragging a box around a set of shapes. Add to the selections using the key.

Select All

Click to select all features on the image. Selection will include all shapes, labels, and any annotation you may have added.

Deselect All

Click to deselect all selected features on the image (or click the image anywhere outside the selected features).

Invert Selection

Select a feature, then click to deselect the feature you selected and select everything else on the image.

Cut

Select the feature (or features) that you want to cut, then click .

Copy

Select the feature (or features) that you want to copy, then click .

Paste

Place the cursor where you want to place the copied or cut feature (or features), then click .

Delete

Select the feature (or features) that you want to delete, then click .

Duplicate Across Channels

Select the feature(s) that you want to duplicate on other channels, then click . This copies the selected features on one channel and pastes them on all other channels.
