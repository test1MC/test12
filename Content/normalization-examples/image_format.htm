Image Format

Do not quantify lossy format images, such as JPEGs.

Some image types undergo a software process known as “compression” when they are saved. This process makes the file size smaller, but it also results in irreversible data loss. To ensure you are analyzing the best available data, do not analyze lossy images, such as JPEGs.
