Analysis Tab

The is used to create a general analysis for any image.

  • The has a drop-down menu for specialized types of analysis. Select the type of analysis to apply and the for that application is opened. The last used parameters for that particular analysis type are used to create the new analysis.
  • The has tools for adding basic shapes to the image. The and tools are not available in multi-image mode. To use these Create tools, change the view to single image mode. All other Shape tools are available in multi-image or single image mode.
  • The has tools for editing the shapes.
  • The tool moves the selected shape(s) to surround a nearby area(s) of pixel intensity.
  • The has tools for defining the background.
  • The options determine what is displayed on the image.
