Clean Box Operation

  1. Open the Clean Box and clean the imaging bed surface with 0.5% as described above. To open, swing the latch down 90° until horizontal, rotate 180° counter clockwise, and swing the latch up before lifting the Clean Box lid.
  2. Close the Clean Box, place the Clean Box on the Pearl Docking Station and lock it in place.
  3. Turn on the docking station and wait for the heater plate to reach the temperature set point (green “running” icon is ON constantly).
  4. Note: Alternatively, the heater plate can be warmed by locking the Clean Box in the Pearl imaging drawer until the set point temperature is reached (see the Pearl Imager Operator's Manual)(see Setting the Heater Plate Temperature on page 1).
  5. Make sure the nose cone plug is in place and start the flow of anesthesia gas as described in Docking Station Operation on page 1. Observe all cautions for the use of isoflurane gas. Wait about a minute for the system to be primed with isoflurane gas.
  6. Unlatch and open the Clean Box lid, remove the nose cone plug, and place the plug in its holder. Remove the mouse from the anesthesia induction chamber and slide the muzzle of the mouse into the nose cone (see the Pearl Imager Operator's Manual)(see Placing Mice on the Imaging Bed on page 1). Make sure the surface is dry before placing a mouse on the imaging bed.
  7. IMPORTANT: Observe the tail of the mouse and make certain the tail will not be pinched between the Clean Box lid and imaging bed when the lid is closed.
  8. Close and latch the Clean Box lid, then install the Clean Box cover on the Clean Box to prevent touching the window or spotting the window with liquids.
  9. If using the Pearl Docking Station, undock the Clean Box and move it to the imaging drawer and lock the Clean Box into the imaging drawer.
  10. IMPORTANT: Move the Clean Box quickly. Anesthesia gas does not flow while the Clean Box is undocked.
  11. Remove the Clean Box cover to expose the window, close the imaging drawer, and acquire an image as described in the Pearl Imager Operator's Manual.
  12. After acquiring images, open the imaging drawer, undock the Clean Box and move it back to the mouse holding facility or recovery chamber where the mouse can be removed.
  13. When the imaging session is complete, replace the Clean Box cover to protect the Clean Box window during storage.
