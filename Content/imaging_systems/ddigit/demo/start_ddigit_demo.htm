This guide will show you how to clean and package the scanner for return to LI-COR. The included Quick Start guide will tell you how to set up and use the scanner. Before you begin, please note the following.

Save all packing materials.

You will need the packing materials to send the scanner back to LI-COR.

Do not send SYBR® Safe stain back to LI-COR.

Any leftover stain is yours to keep!

Cleaning and Packaging Instructions

Thank you for taking the opportunity to try out the (model number 3500). Once you are finished testing, there are four steps to return the test scanner to LI-COR.

  1. Clean the scanner.

  2. Lock the scanner.

  3. Package the scanner.

  4. Send the scanner back to LI-COR.

Specific instructions for each of these steps are located in this guide.
