Gray Mode

The Gray Mode setting allows you to specify the grayscale mode that will be applied the next time you scan a gel. Grayscale images use shades of gray to represent gradations of color.

  • Gray (8-bits) This mode saves grayscale images with 8 bits per pixel. Approximately 256 shades of gray are available in this mode.

  • Gray (16-bits) This mode saves grayscale images with 16 bits per pixel. Approximately 65,536 shades of gray are available in this mode.

    When the Gray 16-bit mode is chosen, images can only be saved in the TIFF format.
