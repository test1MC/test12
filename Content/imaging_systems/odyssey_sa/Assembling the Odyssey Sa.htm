Assembling the Odyssey Sa System

Since this chapter is not an installation guide, it is assumed the computer is setup according to the manufacturer’s instructions. Use the Windows® control panels to make sure the Date and Time control panel is set accurately, and that the Power Options control panel has all power management options set to Never. Set the network configuration to ‘Obtain an IP address automatically.’

  1. Start with both the computer and instrument turned off.
  2. Connect the included Cat. 5 RJ45 cable to the network port on the . The network port is identified in Chapter 2.
  3. Note: Use only the networking cables provided with the Odyssey Sa Imager.
  4. Connect the other end of the Cat. 5 RJ45 cable to the network port on the computer.
  5. Turn on the computer and the , and wait for their start-up procedures to be completed.
  6. Note: Make sure the networking cable is connected to the and computer before turning on the power to either one.
  7. Start on your computer. When starts, it goes through an auto-discovery process that finds the Odyssey Sa instrument (can take several minutes).
  8. To confirm that can communicate with the Odyssey Sa Imager, check that the buttons on the Acquire ribbon are active and the status submenu in the ribbon says Ready.
  10. The instrument will be listed by serial number if connected. Select it and click OK . If it’s not listed, click on Auto Discover to run the autodiscovery process again. It may take several minutes for the Odyssey Sa Imager to establish its IP address on startup.
  12. Open the Application menu, hover over Instrument, and click Connect. In the Scanner Login window, choose your scanner, enter the name and password of an account with Administrator access rights, and click OK.

If you cannot connect to the instrument, see "Reset to Automatic IP Addressing" on page 1.

The default administrator account shipped with the Odyssey Sa instrument is named “admin” and the password is also “admin” (case sensitive). If you can log in using the default admin/admin account, you are successfully connected and will be able to create new user (control) accounts.

For best security, change the password for the “admin” account and keep a record of the new password in a safe place.

Adding Users

User account information is stored on the rather than on the computer. If you do not have a user account, you will need to create one.

  1. Open the Application menu, hover over Instrument, hover over Management, and click Administration.
  2. Click Manage Users to set up a new user account.
  3. Close the System Administration window when you are through adding user accounts.

Setup is now complete.

Configuring Automatic IP Addressing

The Odyssey Sa Infrared Imaging System can be set to an automatic or manual IP address in the Image Studio software in the System Administration window (see step 7 above). Click Edit Network Config...

In this dialog set the network configuration to either Automatic or Manual. In most cases, use Automatic addressing for the .

Reset to Automatic IP Addressing

If you cannot connect to your Odyssey Sa Imager, it is possible that it is configured to an unknown manual address. It is necessary, in this case, to reset the network configuration to automatic addressing by following the steps below.

  1. Start with both the computer and instrument turned off.
  2. Make sure the included Cat. 5 RJ45 cable is connected to the network port on the Odyssey Sa instrument. The network port is identified in Chapter 2.
  3. Note: Use only the networking cables provided with the Odyssey Sa Imager.
  4. Make sure the other end of the Cat. 5 RJ45 cable is connected to the network port on the computer.
  5. Turn on the computer.
  6. Press the power button on the . It will start blinking to show that the instrument is booting.
  7. Press and hold the eject button during the boot process when the green scanning status light is blinking. Make sure the eject button is depressed at the end of the four times that the green scanning status light blinks.
  8. The networking is now configured to automatic addressing and you can release the eject button.
  9. Start on your computer.

The goes through an autodiscovery process and will now be able to find the (can take several minutes).
