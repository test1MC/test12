Assembling the Odyssey Sa System

Introduction

Though not intended to be an installation guide, this chapter gives basic information on assembling the Odyssey Sa Infrared Imaging System. The Odyssey Sa Imager is a network server device that connects to a network, and ultimately to a computer, via TCP/IP. The network configuration of the Odyssey Sa instrument can be set to automatic or manual. An autodiscovery process will discover the Odyssey Sa Imager, allowing the Image Studio software running on a computer to establish communication without any user configuration.
