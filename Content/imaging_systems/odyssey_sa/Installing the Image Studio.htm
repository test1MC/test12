Import the Odyssey Sa Express Software Key

The Odyssey Sa Express software allows the user to setup multiple scans using an automated plate stacker, such as the BioTek® BioStack™ plate stacker (see Installing a BioTek® BioStack™ Plate Stacker on page 1). The Odyssey Sa Express software CD is purchased separately. If you have purchased the Odyssey Sa Express software, follow the directions below to add the software to Image Studio.

  1. Insert the Odyssey Sa Express software key CD.
  2. Click the Application Menu Button and click Import Key.
  3. Browse to the Odyssey Sa Express Software key file on the CD and click Open.
  4. Click OK in the pop-up window and restart Image Studio software.
