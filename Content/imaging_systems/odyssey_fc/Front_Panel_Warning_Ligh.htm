Front Panel Warning Light Indicates an Error

When the is unable to complete a task requested by , or some hardware failure is encountered, the red error indicator on the Odyssey Fc front panel (Figure 1‑1 and Figure 1‑2) flashes and an audible alarm sounds. This error condition is automatically cleared after a few seconds and a message dialog is displayed in the Image Studio software to explain the error. The error message will instruct the user on any further required actions.
