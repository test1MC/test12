Image Studio 5.2 Upgrade

contains fixes for bugs in Image Studio 5.0.

If you are currently using Image Studio 5.0, please upgrade all installations of Image Studio (lite and full) you and your group use from 5.0 to 5.2.

Why is the upgrade recommended?

Two primary reasons:

  1. Image Studio 5.0 contained a bug that sometimes prevented Work Areas from previous Image Studio versions from opening properly. fixes the bug.

    Once a Work Area has been opened in 5.2, do not try to open it in any previous version of Image Studio. This is one reason you should upgrade all your installations of Image Studio to 5.2.

  2. Image Studio 5.2 fixes a bug in Image Studio 5.0 relating to Zip File exports. If you exported a Zip File without first saving changes, your changes would be included in the exported Zip File but not saved to the Work Area.
 How do I upgrade?

When you upgrade to Image Studio 5.2, please upgrade all installations of Image Studio (lite and full) you and your group use.

  • Image Studio Lite: You can upgrade now by visiting the download page.
  • Image Studio Full: If you are a new Image Studio 5.0 user or were eligible for the 5.0 upgrade, please look for an email with a link to the download page.
 How do I transfer images and data between Image Studio 5 and earlier versions?

To transfer images and data between and previous versions, use the and zip file function.

Zip Files exported from Image Studio 3, 4, or 5 can be imported into Image Studio 3, 4, or 5.

  • For , see the Exporting zip files or Importing zip files topic.
  • For Image Studio 4.0, see the Export or Import topics in the Image Studio 4.0 help.
  • For Image Studio 3, click File Management in the left-hand table of contents in the Image Studio 3 Help to read about Import Images and Copy Images.
