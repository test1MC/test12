Frequently Asked Questions

When I update to Image Studio 5.2, why do I need to update all versions of Image Studio that my group and I use?
  1. Most importantly, Image Studio 5.2 provides fixes for bugs in Image Studio 5.0. If you are currently using 5.0, please upgrade to 5.2.
  2. No matter which version of Image Studio you upgrade from, you should upgrade all installations of Image Studio (lite and full) you and your group use to 5.2 because 5.2 uses a new data format. Once a Work Area has been opened in Image Studio 5.2, do not attempt to open it in an earlier version of Image Studio (even 5.0).

You can always export one or more images in an Image Studio zip file to transfer images and analysis data between versions of Image Studio.

For more details about updating to version 5.2, please see the Image Studio 5.2 Upgrade topic.

 Should I use a Manual Analysis or a Western Analysis?

If you only need to quantify a gel or blot with a few, well spaced shapes, a manual analysis may be all you need.

Use a , , or MPX Western Analysis to:

  • Perform an analysis involving molecular weight markers or normalization.
  • Reduce the amount of manual effort required to analyze gels and blots, especially gels and blots that contain many lanes and bands.
 What is signal and how is it calculated?

Signal = Total - (Background x Area)

Signal is the sum of the pixel intensity values (total) for a shape minus the product of the background and the area of the shape.

More detailed information is available in the Odyssey Imaging System Data Quantification technical note.

 Why do analysis shapes on an image not have to be the same size?

The background is multiplied by the area to correct the background value for the size of the shape being quantified, so the size of the shape does not need to be the same for each feature on an image.

 Which background subtraction method should I choose?

The best method for background correction depends on the image and its background consistency. Background must be subtracted to accurately calculate signal from the shape(s) of interest, especially if background values vary throughout an image.
