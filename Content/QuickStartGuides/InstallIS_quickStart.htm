Mac OS X 10.9 (Mavericks) or 10.10 (Yosemite)

  1. Place installation CD into drive.
  2. If installation does not start immediately, browse to the CD and double-click on .
  3. Click Agree to accept the license agreement.
  4. To install, drag the Image Studio icon into the Appli- cations folder (Figure 1).

Install via Download

  1. Go to your Mac System Preferences and click Security and Privacy to open the Security and Privacy window.
  2. Click the lock symbol in the bottom left corner of the window to allow editing, if necessary.
  3. Under the General Tab, enable the Mac App Store and identified developers radio button under the Allow applications downloaded from option.
  4. Download, and double-click on the file on your computer.
    1. Click Agree to accept the license agreement.
    2. To install, drag the Image Studio icon into the Applications folder (Figure 1).
    3. Open the Image Studio Application by double-clicking on the Image Studio icon .

    4. If a window appears that says “ImageStudio.app is an application downloaded from the internet. Are you sure you want to open it?” click Open.

      For a new installation and when updating from a previous version of Image Studio software, you will need to separately import any additional application keys that you may have received, to utilize their functionality.
