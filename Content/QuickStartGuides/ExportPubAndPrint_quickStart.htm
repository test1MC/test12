    To export an image for use in a journal publication or other print media, click Image for Print.

    Click .

    If the exported image resolution does not meet your specific requirements, you can adjust the image quality by setting Size Priority to Inches. Click Reset Inches to en sure the file is in its original size or if a larger image is required, adjust the output size accordingly.

    Using Additional Options under the export menu, you can also export:

    1. The current image using custom settings.
    2. Multiple images as separate files.
    3. Color bar of the current image.
    4. Multiple images in the images table as a single composite image.
    5. Multiple images as a movie

If you want the entire image to be exported and only a portion of image is shown, Cancel out of the Export window, click on Image ribbon, and deselect Apply in the Print/Export group.
