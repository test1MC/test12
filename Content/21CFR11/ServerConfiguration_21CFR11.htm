Server Configuration

Server Hardware

requires server class hardware with built-in redundancy.

  • Modern server class CPU
  • Minimum 8 GB RAM (16 GB recommended)
  • 2 TB disk space

Server Software

  • Windows Server 2012 (English language)
  • Java 1.8.X
  • PostgreSQL 9.3.4
  • Image Studio Server 5.3.X

  • Backup software - Utilize corporate backup system.
