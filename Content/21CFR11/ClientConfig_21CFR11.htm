Client Configuration

A configuration file exists on the client that will configure the client to communicate with the server's IP address/port.

The requires the same minimum hardware configuration as the full version of Image Studio Software.

Hardware and Software

Hardware and Software

  • Minimum 4 GB RAM
  • Minimum display resolution of 1920 x 1080
  • Windows 7 or Windows 8
  • Network Interface Card
  • Image Studio 5.3.x (includes private Java installation ver 1.8.x). Also installs mDNSResponder and USB drivers.
  • Must have IPv6 enabled for instrument communications
  • A second Network Interface Card is required for optimal instrument communication when the instrument is not located on the local network.
  • Must have Bonjour for most instrument auto-connection (mDNSResponder installed with Image Studio Client)
  • Some client event and error log and temporary storage on C:\users\<windows login name>\.licor\Image Studio\temporary. NO image data, analysis data, user log or sample log data stored on client drives.
  • Image Studio Lab Books, Excel spreadsheets, and publication images can be exported to local or other network drives. Exported files are outside of control.
