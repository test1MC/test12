About User Roles

A is a set of allowed actions. are operations performed by a user within . are assigned to users to control which a user can perform.

Only the can add or modify .

  • The allowed actions for default can be adjusted and custom roles can be added.
  • Roles can be assigned overlapping allowed actions.

  • Users can be assigned multiple roles.

    As long as an action is allowed for one of a user's multiple roles, the user may perform that action.

  • Users cannot approve their own work, even if a user is assigned a with the allowed actions to submit and approve.
