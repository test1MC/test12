Using the Profiles Tab to Inspect a Lane Profile

When a lane from a or analysis is selected, the pixel intensity profile throughout the lane can be seen in the .

If is selected as the background subtraction method in the list , a line representing lane background can be displayed beneath the lane profile (see below)(see Figure 1).

. A lane profile displayed in the is shown with background displayed and background subtracted.

  • To display the lane background line, select the check box and a light colored line representing the background will appear beneath the original lane profile.
  • To display the lane profile with background subtracted, select the check box and the graph will be redrawn with lane background subtracted. The signal has already been calculated with the background subtracted, even if has not been selected.
