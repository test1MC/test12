Export Color Bar Only

The check box next to in the must be enabled for a to be exported.

When exporting linked images using , the option to export the color bar as a separate file does not work.

The color bar from an image can be exported by itself to compare images in a presentation, publication, or lab notebook.

  • In the dialog, the font size, bar width, bar length, and horizontal or vertical orientation of the bar can be adjusted.
  • Use the button to find the appropriate location to export the color bar.
  • See "Naming the Exported Image Using the File Name Text Field" on page 1 for how to edit the name of the exported file.
