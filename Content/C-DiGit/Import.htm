Import

The Import menu has choices for importing images and acquisitions from the C‑DiGit Blot Scanner application, as well as from Odyssey® or Pearl® images.

You can import acquisitions created by Image Studio, Odyssey® or Pearl® software, or import other TIFF files for analysis.

Click here to watch a video that demonstrates how to import images.

  1. Open the Application menu and choose Import to display the choices.

Image Studio

Image Studio copies an image acquisitions folder, ZIP file, or Work Area created in Image Studio to the current default images folder. The entire folder is copied, including images, analysis files, and other text files. This is useful if you have previously transferred data to a different location for archiving or you are transferring data from another Work Area.

  1. Choose Image Studio from the Import submenu, and then click Acquisition Folder (or Zip File or Work Area). The Import Acquisitions dialog (below) is displayed.
  3. Navigate to a source folder, zip file, or Work Area and select it. Multiple folders can be selected.
  4. Click Open. The entire folder is copied to the current default directory and the original folder is left in place.
  5. Note: Images cannot be imported from the current Work Area. Be sure to select a folder outside the current Work Area.

Odyssey

Odyssey imports one or more images, scan folders, or projects created with the Odyssey Infrared Imaging System Software into the current default directory. The import creates a new folder with appropriate files. This does not import analysis data.

  1. Choose Odyssey from the Import submenu, and then click TIFF Files, Scan Folder, or Project. The Import Odyssey Image(s) dialog is displayed.
  3. Navigate to a source folder and select images, scan folders, or projects to be imported into the default folder on the file system.
  4. Select the image(s) to import (when TIFF Files is selected) and click Open. You can select one or more folders that contain several TIF files from multiple channels. Note: The files must be the same size and resolution.
  5. You can also select an individual TIF file, or select two TIF files of the same image in different channels. Note: Do not select multiple TIF files that are not of the same image.
  6. The new TIF files are converted upon import. The original TIF files are not changed. The “Name” field inherits the Odyssey “acquisition Name” if a folder is imported. The Image ID field for the new image is assigned a new unique Image ID.

Pearl

Pearl imports one or more TIF files or Work Folders created in Pearl Software.

  1. Choose Pearl from the Import submenu, and then click Acquisition Folder or Work Area. The Import Acquisition(s) dialog (below) is displayed.
  3. Navigate to a source folder and select images to be imported into the default folder on the file system. Click Open.
  4. The “Name” field inherits the Pearl ImageID if a folder is imported. The Image ID field for the new image is the same as the imported ImageID of the folder or Pearl Images.

Other Image

  1. Move the mouse over Other Image to see the options Files and Folder. Click Files to import .tif, .jpg, or .png files. Click Folder to import a folder containing image files.
  2. Navigate to a source folder and select the file or folder containing files to be imported. Click Open.
  3. Note: If the file cannot be imported, an error message will appear at the bottom of the Image Studio window. If the data are inverted during import, Data Inverted During Import appears in the Image Modifications column.
  4. The imported file(s) appears in the Images Table.
