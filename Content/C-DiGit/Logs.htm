Logs

The Logs menu contains choices for printing and/or exporting the application log files. The log files are simple text files that serve as diagnostic records of the Image Studio sessions. A new log file is created each time Image Studio is started; the file name reflects the date during which the file was created.

Print

Opens the Select Log File(s) to Print dialog, where you can choose log files to send to the printer. Highlight a single file, or multiple files. Click Open to send the selected files to the default printer.

Export

Opens the Zip Application Log Files dialog (below), where you can select the log files to be compressed (zipped), and the location to which to save the zip file. The zip file is automatically named EventLogs.zip.
