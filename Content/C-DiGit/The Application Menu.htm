The Application Menu

The Application button is the round button with the Image Studio icon at the upper left corner of the application.

Press the Application button to display the Application menu (below). The leftmost panel displays the primary menu. If the menu item is followed by an arrow symbol (>) click the item to display a secondary menu in the rightmost panel. The secondary menu displays additional choices and their descriptions.

Note: To access these commands with the keyboard, press the ALT key to display the Key Tips.

The footer area of the Application window contains two buttons: Preferences and Exit. Preferences are described later in this section. Click Exit to close the Image Studio application.
