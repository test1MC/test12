Tables

Image data are stored in the folder selected as the Work Area when Image Studio is opened. The data are easily accessed in the Tables at the bottom of the main window. In the Images Table, each row represents one image. Various fields in that row contain identifying information about the image. These fields are grouped as columns that can be moved, added, deleted, sorted, or filtered.

Images Table

  1. Click the Images tab at the bottom of the main window to display the Images Table. Click the double arrow to hide the Table from view.
  3. Mouse over any row to display a thumbnail of that image.
  1. Click any cell to select the entire row and display that image in the window.
  2. Click the corner button to select all of the images or press Ctrl+A on the keyboard.
  3. Right-click any cell to view a menu with options that include copy an image, delete images, or view the properties of an image.
  4. To move a column, drag the column header to the new location.
  5. The fields in each row contain information about the image and are organized as columns across all of the images. Some of the columns are properties of the image and cannot be edited. Examples of these columns include Image ID, Acquire Time, Channels, Resolution, and Instrument Name. Other columns can be edited with information for that image.
  6. To edit a field, double-click on the cell so that a cursor appears. If a cursor does not appear, that field cannot be edited.
  7. Add text where the cursor appears. Press Enter when complete.

Note: To easily find images later, enter identifying information in one or more fields after acquiring an image.

Add or Remove a Column

  1. Click the Columns button at the top right of the table.
  2. To add an Experiment column, for example, enable the check box, indicating that column will appear in the table.
  3. To remove the Resolution column, for example, disable the check box and remove that column from the table.
  5. Click Save. The column will be added to the table.

Edit the Image Title

  1. Click the Columns button to view the Image Columns dialog.
  2. Click Use an additional image column with the Image ID to enable the check box.
  4. Click the down arrow in the selection box to choose a column to add to the Image ID to form the Image Title.
  5. Click Project. Text in the Project column will be added to the title of every Image.
  6. Click Save.

Add Custom Columns

  1. Click the Columns button to view the Image Columns dialog.
  2. In the Custom Columns menu, replace Custom Image Column 1 with Department.
  4. Click Save.
  5. To add this custom column to the table, click the Columns button to view the Image Columns dialog again.
  6. To add the Department column, click the column description to enable the check box.
  7. Click Save. The Department column will be added to the right of the existing columns.

Paste Data in a Column

  1. Double-click the first cell in the Department column so that a cursor appears in the cell.
  2. Type ‘Tutorial’ at the cursor position using the keyboard.
  3. Drag the mouse over ‘Tutorial’ to highlight it. The cursor should still appear in the cell.
  4. Click the Copy button, or Ctrl+C on the keyboard to copy the data to the clipboard.
  5. Drag the mouse over the cells in the Department column (or press the Ctrl key while clicking each cell once) to select which cells to fill with ‘Tutorial’.
  6. Click the Paste button, press Ctrl+V on the keyboard, or use the right-click menu to paste the data in the selected cells.

Paste Data from a Spreadsheet to a Column

  1. In a spreadsheet, select data and click the Copy button, or press Ctrl+C on the keyboard to copy the data to the clipboard.
  2. In the Table, drag the mouse over the cells in the Department column (or press the Ctrl key while clicking each cell once) to select which cells in a column to place the data.
  3. Click the Paste button, press Ctrl+V on the keyboard, or use the right-click menu to paste the data in the selected cells.

Copy Data to a Spreadsheet

  1. Drag the mouse over the rows or press the Ctrl key while clicking on each row once to select which rows to copy. To copy all rows, press Ctrl+A on the keyboard.
  2. Click the Copy button, press Ctrl+C on the keyboard, or use the right-click menu to copy the data to the clipboard.
  3. In a spreadsheet, press Ctrl+V on the keyboard to paste the data in the selected cells.

Note: The column headings will be pasted above the data.

Sort by Column

  1. To sort by any column, click once on the header. The table will be sorted by that column. Note: No change will be made if the fields in a column are all the same.
  2. To change the direction of sorting, click the small box in the header with the up or down arrow. Or, right-click on the header and select Sort Ascending or Sort Descending.

Filter the Table Manually

  1. To manually choose images, drag to highlight the rows to select the desired images. Or, select the desired images by pressing Ctrl on the keyboard and clicking once on each row.
  2. Click the Filter button.
  3. Click Selection to show only the selected images in the table. The Filtered button in the notification area will be highlighted.
  4. Click Display Filtered to toggle between showing only the selected images and showing all images in the table.
  5. To remove the filter, click Clear Selection to remove only the Selection filter, or Clear All to remove all filters currently applied.

Define Filters

One or more defined filters can be applied to the table to show only the images of interest.

  1. Click the Filter button.
  2. Click Column to view the menu of all of the column headings. Select Sensitivity to filter using the Sensitivity column.
  4. Note: Right-click on any column header in the Table to drop down a menu for that column.
  5. Deselect High by disabling the check box.
  1. Click OK. Only images with Shapes with a Standard value in the Sensitivity column will appear in the Table.

For more Filter Options…

  1. Click the Filter button. Click Column to view the menu of all of the column headings. Choose the column that you want to filter.
  2. Click Define Filter…
  3. Click one of the filtering phrases (e.g. Equals) to open the Column Filter Sensitivity dialog.
  4. Select filtering phrases from the drop-down menu by clicking on the arrows next to each box, or type your own phrase.
  6. Click OK.

Note: If you close Image Studio, your filtering criteria are preserved.

View the Shapes Table

  1. Click the Shapes button at the bottom of the main window to view the data for each shape.
  3. The default columns are defined below.
    1. Image Name - Original acquisition name
    2. Channel - Chemi
    3. Name - Shapes are assigned a number in sequential order. Double-click on the number in the column to edit the shape name.
    4. Signal - The Signal is the sum of the individual pixel intensity values (Total) for a shape minus the product of the average intensity values of the pixels in the background (Bkgnd.) and the total number of pixels enclosed by the shape.
    5. Total - Sum of the individual pixel intensities for the shape
    6. Area - Total number of pixels enclosed by a shape
    7. Bkgnd. - Designated background subtraction
    8. Type - Signal or Background
  4. Move, add, delete, sort, and filter the columns in the same way as in the Images Table.

Exporting Data

Click the Report button and choose Save As, Options or Launch Spreadsheet to prepare or open a ‘.xls’ file with the selected data.

Export File

  1. Click Options to view the Data Options dialog. Choose the file format (text or XLS) and whether to export all rows or only selected rows.
  3. Click Save As to open the Images Table Export menu.
  4. Select the folder destination. Click Save.
  5. The file will be saved in the selected folder.

Launch Spreadsheet

  1. Click Options to view the Data Options menu (see Step 1 above). Export all rows or selected rows. Click Save. If necessary, select the rows in the table to export.
  2. Click Launch Spreadsheet to open the Launch Spreadsheet dialog.
  4. Select the folder destination and enter a file name. Click Save.
  5. Any application that reads these types of files will open them with the selected data and corresponding column headings.
