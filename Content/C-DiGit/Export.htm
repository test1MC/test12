Export

The Export menu has choices for exporting images and movies. You can choose to export the currently displayed image, or selected images in the Images table. Click the Images tab at the bottom of the main window, and select the images that you want to export.

Click here to watch a video that demonstrates how to export data.

Export Current Image

Open the Application menu. Click the Export menu and expand the Single Image View submenu. Check Current Image. The Export Current Image dialog is displayed.

  1. Select the Size Priority.
  2. Select Inches to set the output size to inches when changing DPI. The output pixel count changes proportionally when the inches value is changed.
  3. Select Pixels to set the output size in pixels when changing DPI. The output inches value changes proportionally when the pixel count is changed.
  4. Click Reset Pixels or Reset Inches to return the Output to the Original Size.
  5. Select the Publication Resolution and the Image Format. If the image is for viewing on a website or for attaching to an e-mail, choose 150 dpi and JPEG or PNG file.
  6. If you accept the default folder and name, the folder will be your current Work Area and the name will be the Image ID with an extension indicating the File type, for example, Image_0000021_01.tif.
  7. If you want to change the default name or folder, click Edit Name to change the name; click Browse to navigate to a different folder.
  8. Click Save when finished.

Single Image View–Exporting Multiple Images in Individual Files

Click the Export menu and expand the Single Image View submenu.

Check Selected Images in Images Table. The Export Selected Images dialog is displayed.

  1. Enable Overwrite existing files, or disable if you don't want the existing files to be overwritten.
  2. Select the Publication Resolution and the Image Format. If the image is for viewing on a website or for attaching to an e-mail, choose 150 dpi and JPEG or PNG file.
  3. If you accept the default folder and name, the folder will be your current Work Area and the name of each image will be its Image ID with an extension indicating the File type, for example, Image_0000019_01.tif, Image_0000020_01.tif.
  4. If you want to change the default name or folder, click Edit Name to change the name; click Browse to navigate to a different folder.
  5. Click Save when finished.

Multiple Image View

The Export Multiple Image View option exports all images in the current Image Table. To export only selected images, use a Selection Filter. Click the Multi-Image mode button, and then open the Application menu. Choose Multiple Image View from the Export menu. The Export Multi Image View dialog (below) is displayed.

  1. Select the Size Priority.
  2. Select Inches to set the output size in inches when changing DPI. The output pixel count changes proportionally when the inches value is changed.
  3. Select Pixels to set the output size in pixels when changing DPI. The output inches value changes proportionally when the pixel count is changed.
  4. Select the Publication Resolution and the Image Format. If the image is for viewing on a website or for attaching to an e-mail, choose 150 dpi and JPEG or PNG file.
  5. Check Export Images Table to include the Images table.
  6. If you accept the default folder and name, the folder will be your current Work Area and the name will be MultipleImageView.
  7. If you want to change the default name or folder, click Edit Name to change the name; click Browse to navigate to a different folder.
  8. Click Save when finished.

Edit File Name

The Edit File Name dialog is displayed if you click Edit Name in the Export Selected Images or Export Current Image dialogs. The Edit File Name dialog provides a way to build a file name using selections from a list of the columns in the Images Table. The selections are inserted into the File Name text field, preceded by a tilde (~).

  1. Select a column field from the drop-down menu.
  2. Click Insert. The field is added to the name in the File Name text field at the cursor location.
  3. Repeat these steps to add more fields. Notice that the fields are preceded by a tilde (~).
  4. To preview the file name, view the Example File Name that is displayed in the dialog as you add fields. Hint: Separate inserted fields with an underscore or space character to make the fields readable.
  5. Click OK when finished.
  6. Note: If you are exporting multiple images, be sure to include a unique value (e.g. Image ID). If you simply specify a file name, such as "My_Image~Date", as each file is created it will use that name, thus overwriting the previously saved file.

Export Image Studio

The entire folder is copied, including images, analysis files, and other text files. This is useful if you are transferring data to a different location for archiving or you are transferring data from another Work Area. You can export the files into a folder of your choosing, or compress the files into a single .zip folder.
