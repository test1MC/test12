Preferences

The Preferences dialog has four page views: General, Image LUTs, Image View, and XLS Export. Click the labels under ‘Preference List’ to change the page view.

General

Use the General preferences to control general application features. You can set the undo limit, choose whether to show tooltips, specify your organization name and company logo to display on imported images, set the quantification format and the number of digits to display, and choose the image size units (inches or centimeters).

  • The Undo Limit sets the number potential undo operations that can be stored by the application.
  • When enabled, Show Tooltips displays a tooltip when the cursor is hovered over an icon or label in the application.
  • The Organization Name specifies the organization name to be used in an Images Table column.
  • The Quantification Format specifies whether to use Scientific or Numeric notation for floating point values. Note: Excel reports numeric formats using its default format for floating point fields. These fields my be displayed as both standard and scientific formats unless the formatting is overridden in Excel.
    • Use the counter to specify the number of significant digits to display.
  • The Export Image Size Units sets the size units to Inches or Centimeters.

Image Display

Click ImageDisplay to specify the default image display colors for new acquisitions.

Image View

Contains a check box that allows you to enable or disable the use of the mouse scroll wheel to zoom in or out on the image view. When the cursor is placed over the image in the main Image Studio window, enabling the check box allows you to enlarge or reduce the image using the mouse scroll wheel.

XLS Export

Specifies how saturation (infinity) and non-numeric values (NaN) are expressed when exporting into an Excel (.xls) spreadsheet file.
