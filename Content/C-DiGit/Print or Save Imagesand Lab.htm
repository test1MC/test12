Print or Save Images and Lab Books

Print an Image

  1. Click the Image Studio Application Button, hover over Print, and choose your printer and/or page setup parameters, if desired, with the Page Setup and Select Printer options.
  2. Click the Image Studio Application Button, hover over Print, and click Single Image Per Page to send the currently displayed image to the default printer in the currently defined Lab Book format.

Save Image(s) as a PDF File

  1. Click the Lab Book tab to open the Lab Book ribbon.
  2. Choose Single Image Per Page or Multiple Images Per Page in the Layout group.
  3. Click Save to save the current image(s) in .pdf format.
  5. Alternatively, you can click Print Preview in the PDF group, and then click the Save As PDF button at the top of the window to save the current image(s) in .pdf format.

Print a Lab Book

In the Lab Book ribbon, select Shapes Lab Book in the Layout group and click Print Preview, and then click Print to print the default report for the current image.

Click here to watch a video that demonstrates how to print lab reports.

To create a custom report, click Edit in the Layout group to open the Lab Book Layout Editor dialog (below).

  1. Items that are available to add to the report appear under Available in the right- hand column. Select an item and click Add to move it to the In Use column on the left.
  2. Click OK to include the items in the In Use column in the report.
  3. Click Print Preview, and then click Print to print the custom report for the current image(s).

Save a Lab Book

In the Lab Book ribbon, select Shapes Lab Book in the Layout group and click Save to save the default report for the current image.

To create a custom report, click Edit in the Layout group to open the Lab Book Layout Editor dialog (below).

  1. Items that are available to add to the report appear under Available in the right- hand column. Select an item and click Add to move it to the In Use column on the left.
  2. Click OK to include the items in the In Use column in the report.
  3. Click Save to save the custom report for the current image(s).
