Acquire Ribbon

Click the Acquire tab to display the Acquire ribbon options. The Acquire ribbon is used to set the parameters for capturing the image from the membrane on the C‑DiGit scanning surface.

Status

Mouse over the C‑DiGit logo to display the C‑DiGit Instrument Status bar, which shows the serial number and instrument software version of the instrument that is currently connected (below). If an instrument is not connected, the Status display will show ‘Not Available’.

Setup

No Analysis – An analysis is not performed after the image acquisition is finished. Automated analysis options are not available with C‑DiGit.

Image Table Info – Opens a dialog where you can enter initial values for the image table columns (below).

The values entered in the dialog above are used for all new images created during the current Image Studio session; this automates data entry for the table headers so that you do not need to re-enter the information for each image.

Sensitivity

Sets the Sensitivity for the image acquisition, where an image acquired using Standard sensitivity requires about 6 minutes, and an image acquired using High sensitivity requires about 12 minutes.

Scanner

Click Start to begin the image acquisition. Click Cancel at any time to stop the image acquisition and discard the image.
