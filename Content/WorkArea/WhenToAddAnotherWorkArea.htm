When to Add Another Work Area

  • A user may choose to create more than one to separate different projects, or if one is so full that it takes too long to open.
  • When dealing with projects that require different settings and analysis types, keeping a for each project may be expedient, because Image Studio will recall the last used settings specific to each project when that project's is opened.
  • Data can be moved using the operating system's file manager to prevent making a new . Moving and archiving unused data may be a better choice than creating a new .
