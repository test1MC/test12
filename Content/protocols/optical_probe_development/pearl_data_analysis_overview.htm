Data Analysis

Single images can be screened one-by-one or in a series using the Link Lookup Table (LUT) tool in Image Studio. The LUT tool sets the display values for all images in a series to the display values of a selected image.

See the Image Studio help for more details.

Image series that work well with this tool include:

  • Dose responses
  • Time courses
  • Clearance profiles
  • Biodistributions

To link the display values for a series of images in Image Studio:

  1. While holding down CTRL, click all the images in the Images Table that you want to link.

  2. Above the Image Table, click Filter > Selection.

  3. The Image Table will now show only the images you selected.

  4. In the Images Table, click the image whose display values you want to use.

  5. Click Link at the top of the Display Panel.
